# Security Policy

## Supported Versions

We actively support the latest version of `sunat-utils`.

| Version | Supported |
|--------|-----------|
| 1.2.x  | ✅ Yes |
| < 1.2  | ❌ No |

---

## Reporting a Vulnerability

If you discover a security issue, please do NOT open a public issue.

Instead, report it privately via:

- GitHub Security Advisories
- Or repository maintainer contact email

---

## What to include

Please provide:

- Description of the issue
- Steps to reproduce
- Possible impact
- Suggested fix (if available)

---

## Response time

We aim to respond within 48–72 hours.

---

## Scope

This security policy applies only to the `sunat-utils` package and its official source code.
